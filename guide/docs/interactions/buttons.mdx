# Buttons

Components allow users to interact with your bot through interactive UI elements. In this section, we will discuss how bots can receive and process information from these elements through the library.

<br />
<DiscordMessages>
	<DiscordMessage profile="bot">
		<div slot="interactions">
			<DiscordInteraction profile="user" command={true}>
				buttons
			</DiscordInteraction>
		</div>
		Need help?
		<div slot="actions">
			<DiscordButtons>
				<DiscordButton type="success">Yes</DiscordButton>
				<DiscordButton type="danger">No</DiscordButton>
				<DiscordButton type="secondary">Maybe?</DiscordButton>
			</DiscordButtons>
		</div>
	</DiscordMessage>
</DiscordMessages>
<br />

## Button styles

| Name      | Syntax                                                                                    | Color   |
| --------- | ----------------------------------------------------------------------------------------- | ------- |
| Primary   | `disnake.ButtonStyle.primary` / `disnake.ButtonStyle.blurple`                             | Blurple |
| Secondary | `disnake.ButtonStyle.secondary` / `disnake.ButtonStyle.grey` / `disnake.ButtonStyle.gray` | Grey    |
| Success   | `disnake.ButtonStyle.success` / `disnake.ButtonStyle.green `                              | Green   |
| Danger    | `disnake.ButtonStyle.danger` / `disnake.ButtonStyle.red`                                  | Red     |
| Link      | `disnake.ButtonStyle.link` / `disnake.ButtonStyle.url`                                    | Grey    |

<br />
<DiscordMessages>
	<DiscordMessage profile="bot">
		<div slot="actions">
			<DiscordButtons>
				<DiscordButton type="primary">Primary</DiscordButton>
				<DiscordButton type="secondary">Secondary</DiscordButton>
				<DiscordButton type="success">Success</DiscordButton>
				<DiscordButton type="danger">Danger</DiscordButton>
				<DiscordButton type="link" url="https://guide.disnake.dev/">
					Link
				</DiscordButton>
			</DiscordButtons>
		</div>
	</DiscordMessage>
</DiscordMessages>
<br />

:::note

`Link` buttons do not have a `custom_id`, and do not send an interaction event when clicked.

:::
